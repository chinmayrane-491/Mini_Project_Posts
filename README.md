# Mini_Project_Posts
### We Create a Post Web Application Using Express Js, Node Js, EJS, and MongoDB

### Features :
##### 1) Login (Already Created User can login)
##### 2) Sign up (Create new User)
##### 3) Upload a photo as Profile
##### 4) Like the Post or Unlike the Post
##### 5) Edit the post
